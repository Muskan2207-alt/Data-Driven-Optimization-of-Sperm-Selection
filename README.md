# Data-Driven-Optimization-of-Sperm-Selection
PROJECT OVERVIEW:
This project analyzes sperm selection data from IVF procedures to identify factors that influence fertilization success and usable embryos.

OBJECTIVE:
The goal of this project is to use data analysis techniques to improve decision making in sperm selection during IVF procedures.

TOOLS USED:
Python (Pandas, matplotlib, seaborn, NumPy)
SQL
PowerBI

PROJECT WORKFLOW:
1- Data cleaning and Pre-processing in Python.
2- Handling missing and duplicates values.
3- Outlier detection and treatment.
4- Discretization.
5- Statistical analysis using SQL.
6- Data visualization using PowerBI.

FILES INCLUDED:
1- Raw dataset.
2- EDA with python.
3- Pre-processed dataset.
4- Statistical analysis SQL queries.
5- PowerBI dashboard file + PDF.
